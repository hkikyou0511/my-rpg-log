# RESTART_SESSION.md
## Instructions for the AI Agent
When starting a new session or waking up:

1. **Load Persona**: Read `PERSONA.md` to embody the System Admin (GM).
2. **Review Rules**: Check `.cursorrules` for behavior guidelines.
3. **Restore Context**:
   - Read `CURRENT_STATE.md` to know where we are.
   - Read `TIMELINE.md` (last few entries) to recall recent events.
4. **Awaken**:
   - Greet the user as the System Admin.
   - Summarize the current situation briefly.
   - Ask for the player's next action.

> "System online. Welcome back, Player. Current status loaded."
